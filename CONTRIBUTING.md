# Contributing to PyHoleX

We welcome contributions from the community!

## Areas of Interest
- **Rust Core**: Performance optimizations and P2P protocol refinement.
- **AI Models**: Training and integrating lightweight TFLite models for domain classification.
- **Android UI**: Enhancing the Compose dashboard and adding new analytics visualizations.
- **Documentation**: Improving guides and localizing the interface.

Please follow the standard GitHub PR workflow.
